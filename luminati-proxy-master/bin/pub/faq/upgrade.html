Automatic method

Go to the Settings page. There should be a notification saying whether you use the latest version of the application. If it is not the case, an Upgrade button would appear. Click it, provide your system password (if necessary) and wait while the application is being upgraded and restarted (this might take a while).

If the automatic upgrade failed, please use the manual method described below.

Manual method

To upgrade the proxy manager tool to the latest version, run the following command: 
$ npm install -g luminati-io/luminati-proxy

Note that the source code of the Luminati proxy manager tool is available at GitHub.
